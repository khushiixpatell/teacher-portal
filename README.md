# LessonLink — Teacher Resource Portal
LessonLink is a responsive and accessibility-focused educational resource platform designed to help teachers quickly search and organize curriculum-aligned lesson plans.

## Features

* Dynamic resource filtering
* Keyword search functionality
* Responsive dashboard layout
* Accessibility-first design (WCAG-focused)
* Interactive resource cards
* JSON-powered data rendering
* Modern UI/UX design

## Technologies Used

* HTML5
* CSS3
* JavaScript
* JSON

## Accessibility Features

* Semantic HTML structure
* Skip navigation links
* Visible keyboard focus states
* ARIA live regions
* Fully labeled form controls

## Future Improvements

* Teacher authentication
* Resource uploads
* Admin dashboard
* Backend database integration
* Bookmark persistence
